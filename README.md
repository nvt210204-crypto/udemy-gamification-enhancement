# udemy-gamification-enhancement
BA Case Study - Udemy Gamification Enhancement
## Overview

This project proposes a gamification enhancement for Udemy to improve learner engagement, retention, and course completion rates.

## Objectives

- Increase learning consistency
- Encourage course completion
- Improve learner retention

## Key Features

- Achievement System
- Reward Store
- XP Point Mechanism
- Learning Streak

- BRD
- BPMN AS-IS
- BPMN TO-BE
- User Stories
- Wireframes
